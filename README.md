# Retirement Age Analysis – Power BI

## Description
Ce projet Power BI analyse l’âge de départ à la retraite selon différents profils.
L’objectif est de comprendre les écarts et tendances à partir de données structurées
et de proposer des visualisations claires pour l’aide à la décision.

## Objectifs du projet
- Analyser l’âge de départ à la retraite
- Comparer les profils selon la CSP, le sexe et la carrière
- Identifier les écarts et distributions
- Produire des tableaux de bord lisibles et exploitables

## Données
Les données ont été préparées et nettoyées avec Power Query.
Elles comprennent notamment :
- Âges de départ
- Profils (CSP, sexe)
- Données de carrière

## Modélisation
- Structuration des tables (faits et dimensions)
- Création des relations
- Mise en place des calculs analytiques

## Indicateurs clés (KPI)
- Âge moyen de départ à la retraite
- Écarts par profil
- Distributions et comparaisons
- Analyses temporelles

## Outils utilisés
- Power BI Desktop
- Power Query
- DAX
- Excel
